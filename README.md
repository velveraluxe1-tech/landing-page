# Landing Page

A modern, responsive landing page built from a Figma design. This project features a clean layout with hero section, cards, customer testimonials, and a comprehensive footer.

## Features

- **Responsive Design**: Built with modern CSS and responsive layouts
- **Figma-Accurate**: Matches the original Figma design specifications exactly
- **Modern Typography**: Uses Inter font family with proper weight variations
- **Clean Code**: Well-structured HTML and CSS

## Project Structure

```
landing-page/
├── index.html          # Main HTML file
├── styles.css          # Stylesheet
├── images/             # Image assets
│   ├── hero-image.png
│   ├── card-image-*.png
│   ├── section-image.png
│   └── avatar-*.png
└── README.md           # This file
```

## Design Specifications

- **Colors**: 
  - Primary: #000000 (Black)
  - Secondary: #FFFFFF (White)
  - Text Secondary: #828282 (Gray)
  - Border: #E6E6E6 (Light Gray)
  - Background: #F7F7F7 (Light Gray)

- **Typography**: Inter font family
  - Headings: 64px, 48px
  - Body: 24px, 20px, 16px
  - Weights: 400, 500, 600, 700

- **Layout**: 
  - Max width: 1440px
  - Container padding: 80px
  - Border radius: 8px, 12px

## Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. All images should be in the `images/` folder
